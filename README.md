# CountAll - App de Contadores

Una aplicación Android simple y elegante para gestionar múltiples contadores.

## Características

- **Múltiples Contadores**: Crea tantos contadores como necesites
- **Interfaz Intuitiva**: Diseño limpio con Material Design
- **Fácil de Usar**: Botones + y - para incrementar o decrementar valores
- **Personalizable**: Cada contador puede tener su propio nombre
- **Vista en Cuadrícula**: Los contadores se muestran en una cuadrícula de 2 columnas

## Funcionalidades

- ✅ Agregar nuevos contadores con el botón flotante
- ✅ Incrementar contador con el botón +
- ✅ Decrementar contador con el botón -
- ✅ Editar el nombre de cada contador
- ✅ Eliminar contadores individuales

## Requisitos

- Android Studio Arctic Fox o superior
- Android SDK 24 (Android 7.0) o superior
- Kotlin 1.9.0

## Instalación

1. Clona este repositorio
2. Abre el proyecto en Android Studio
3. Sincroniza el proyecto con Gradle
4. Ejecuta la aplicación en un emulador o dispositivo físico

## Tecnologías Utilizadas

- **Kotlin**: Lenguaje de programación principal
- **Material Design Components**: Para una UI moderna y consistente
- **RecyclerView**: Para mostrar la lista de contadores de manera eficiente
- **ViewBinding**: Para acceso seguro a las vistas
